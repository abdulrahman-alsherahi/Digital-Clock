# Digital Clock in C++

A simple terminal-based digital clock written in **C++**.
The program displays hours, minutes, and seconds and updates the time every second.

## Features

* Real-time clock display in the terminal
* Automatic update every second
* Hour, minute, and second tracking
* Simple and beginner-friendly C++ project


## How It Works

The user enters the starting time (hours, minutes, seconds).
The program then continuously updates the time every second using a loop and time delay.
